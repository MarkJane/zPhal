## zPhal介绍

Open-sourced blog system for building high performance experience base on Phalcon framework.

基于Phalcon开发的开源博客系统。

## 运行环境
### 环境要求
* PHP 7.0+
* Phalcon 3.0.0+
* MySQL
* Nginx or Apache
* Redis
* Composer
